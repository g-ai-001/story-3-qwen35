# 小说规划 - 《双骨至尊》

## 核心设定

**世界观**：九天十地，万族林立。人族居于中央神土，外有异域虎视眈眈。修行之路，以骨为基，以血为引，踏天骄之路，证至尊之位。

**核心线索**：
- 天生至尊骨的石毅被族人强行夺骨，置入堂弟石昊体内
- 失去天赋的石毅以凡体重修重瞳之力，走上复仇证道之路
- 获得至尊骨的石昊，肩负更艰巨的使命
- 昔日兄弟宿敌，最终要在帝关前，面对异域不朽之王的入侵
