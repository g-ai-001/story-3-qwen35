# 第 111 章 - 浩劫之源

混沌祭坛之上，五块混沌碎片终于完全融合，化作一道璀璨的混沌之光，没入石毅与石昊体内。

两兄弟盘膝而坐，周身环绕着混沌之气。石毅的混沌重瞳绽放出耀眼的光芒，重瞳之中仿佛蕴含着整个宇宙的奥秘；而石昊的混沌圣骨则发出铮铮鸣响，至尊法则在骨髓中流转。

“这就是混沌之主的力量吗？”石昊感受着体内澎湃的力量喃喃道。

石毅缓缓睁开眼睛，混沌重瞳扫视着九天十地的每一个角落：“这股力量……比我想象的更加可怕。”

就在此时，远处传来一阵惊天动地的轰鸣声。帝关方向，一道冲天的黑色光柱直冲云霄，恐怖的威压让整个九天十地都为之颤抖。

“那是……浩劫之源的方向！”石毅瞳孔骤缩。

混沌重瞳中映照出帝关外的景象：异域大军再次集结，不朽之王们虎视眈眈，而在他们身后，一个更加恐怖的存在正在缓缓苏醒。

“那是……真正的浩劫之主本体！”石昊面色大变。

石毅站起身来，混沌之力在周身流转：“看来，我们成为混沌之主，只是这场浩劫的开始。”

两人对视一眼，目光中充满了决然之色。不管前方有多少艰难险阻，他们都将携手面对，守护九天十地的和平。

帝关之前，风云再起。混沌之主的名号，将响彻整个宇宙。

（本章完）
