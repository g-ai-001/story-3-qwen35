# 少年至尊骨与万族争雄

## 简介

天生至尊骨的石毅被族人强行夺骨，置入堂弟石昊体内。失去天赋的石毅并未沉沦，反以凡体重修重瞳之力，走上一条极尽升华的复仇证道之路。而获得至尊骨的石昊，则肩负着另一条更艰巨的使命。昔日兄弟宿敌，最终要在帝关前，面对异域不朽之王的入侵，做出守护与牺牲的抉择。

**看点**：双主角视角下的命运纠葛，"夺骨"的极致冲突，兄弟阋墙到携手抗敌的升华。

## 核心设定

### 双主角
- **石毅**：天生至尊骨持有者，被夺骨后觉醒重瞳之力，走上复仇证道之路
- **石昊**：至尊骨新主人，背负原罪，在真相与使命间挣扎

### 世界观
- 荒域三十六国、七十二洞天
- 帝关：抵御异域不朽之王入侵的最终防线
- 异域：不朽之王统领的神秘疆域
