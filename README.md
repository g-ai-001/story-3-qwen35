# 《双骨至尊》

> 双骨降世，天命相克。
> 一朝夺骨，兄弟陌路。
> 他日重逢，是敌是友？
> 帝关之前，谁主沉浮？

## 简介

天生至尊骨的石毅被族人强行夺骨，置入堂弟石昊体内。
失去天赋的石毅并未沉沦，反以凡体重修重瞳之力，走上一条极尽升华的复仇证道之路。
而获得至尊骨的石昊，则肩负着另一条更艰巨的使命。

昔日兄弟宿敌，最终要在帝关前，面对异域不朽之王的入侵，做出守护与牺牲的抉择。

## 核心看点

- **双主角视角**：石毅与石昊，两条命运线交织，演绎兄弟阋墙到携手抗敌的史诗
- **夺骨冲突**：开篇即高潮，极致冲突展现人性善恶
- **复仇证道**：从云端跌落泥潭，以凡人之躯比肩神明
- **守护牺牲**：探讨人性、正义、爱与牺牲等永恒主题

## 世界观

**九天十地**：万族林立，人族居于中央神土，外有异域虎视眈眈。

**修行境界**：搬血境 → 轮海境 → 尊者境 → 圣境 → 帝境（伪帝、真帝、大帝、帝尊、不朽）

**帝关**：九天十地与异域交界之地，抵御异域入侵的最后屏障。

**至尊骨与重瞳**：上古时代，一位无上至尊同时拥有至尊骨与重瞳，横扫九天十地。陨落后，两者分离，成为两种独立的神骨。重瞳是开启帝关的钥匙，唯有重瞳者才能调动帝关力量。

## 主要角色

| 角色 | 身份 | 特点 |
|------|------|------|
| 石毅 | 天生至尊骨（后失去） | 从云端跌落，以凡体重修，复仇证道 |
| 石昊 | 重瞳者 | 被夺重瞳，十年隐忍，一朝觉醒 |
| 石震天 | 石族老祖（已故） | 杀兄夺权，夺骨阴谋的始作俑者 |
